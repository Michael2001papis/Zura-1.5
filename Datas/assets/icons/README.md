# PWA Icons

## אייקונים נדרשים

לצורך PWA מלא, יש צורך בקבצי PNG הבאים:

1. **icon-192.png** - 192x192 pixels
2. **icon-512.png** - 512x512 pixels

## יצירת האייקונים

ניתן ליצור את האייקונים באמצעות:

1. **כלי מקוון:**
   - https://realfavicongenerator.net/
   - https://www.pwabuilder.com/imageGenerator
   - https://favicon.io/

2. **כלי מקומי:**
   - GIMP / Photoshop
   - ImageMagick
   - Inkscape (להמרת SVG ל-PNG)

3. **המרת SVG קיים:**
   - קובץ `icon.svg` קיים בתיקייה זו
   - ניתן להמיר אותו ל-PNG בגדלים הנדרשים

## הערה

כרגע יש SVG זמני. יש להמיר אותו ל-PNG בגדלים 192x192 ו-512x512 ולהציב אותם בתיקייה זו.
